# C-Roulette
Version 1.0

Casino Roulette game develop in C.
Made by: Charles Bournonville and Marius Courquin.

Follow the instructions below to manage or run the game.

## Launch the dealer
```sh
./dealer.sh
```
## Launch a player
```sh
./player "yourname"
```
## Reset the users data
```sh
make reset
```
